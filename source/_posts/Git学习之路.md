#  Git学习之路

鉴于Git在将来的重要作用，今天开始学习Git命令，以便更方便的在IDEA中图形化使用Git命令。

## 1.	创建仓库

```Git
$:mkdir learnGit	//新建一个仓库目录
$:cd learnGit		//进入仓库
```

## 2.	初始化仓库

```
$:git init			//初始化仓库
Initialized empty Git repository in D:/learnGit/.git/
```

此时的该文件夹会有（master）的字样
